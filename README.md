# Yugioh DataBase App
## _Tech Stack and Libraries used_:
   -100% Java
   -Retrofit
   -RxJava
   -RxAndroid
   -MPAndroidChart
   -Glide
   - Architecture
      -MVC Architecture (Model-View-Controller)
      -  View are thrown into standalone classes
      -  Models Act as controllers and communicates with view classes through interfaces
